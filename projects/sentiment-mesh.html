Sentiment Mesh

Each point is a post — mapping AI sentiment across six subreddits in three dimensions.

Type

Weekend Project

Data Source

Reddit (6 subreddits)

Analysis

VADER Sentiment

The discourse around AI is enormous, fragmented, and emotionally loaded. Sentiment Mesh is a weekend exploration into making that discourse navigable — mapping thousands of Reddit posts across six AI-focused communities into a single 3D space.

Each point in the visualization represents a single post, positioned by sentiment score (VADER), recency, and community membership. The result is an immediate, spatial sense of how different AI communities talk about the technology differently — which communities skew optimistic, which are anxious, and how sentiment shifts over time.

Built in a weekend using Python for data collection and analysis, and Three.js for the 3D rendering. A reminder that sometimes the most interesting projects start with a question and forty-eight hours.

Technologies

  • Python
  • VADER Sentiment Analysis
  • Reddit API
